# Finance-data-analytics-project
Data analytics project using Excel and PowerBi to visualize customer spending patterns and sales insights.
## Project Overview
This project focuses on analyzing customer spending patterns using **Power BI** and **CSV data analysis**.  
The dashboard provides insights into customer purchases, product category performance, city-wise revenue, payment preferences, and monthly sales trends.

The objective is to transform raw transactional data into meaningful business insights for better decision-making.

## Tools & Technologies Used
- Microsoft Excel
- Power BI
- CSV Dataset
- Data Cleaning & Transformation
- Dashboard Visualization

## Dataset Information
- **Dataset Name:** Customer Spending Data
- **Rows:** 500
- **Columns:** 13
